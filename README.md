# MongoDB Connector with Mongoose

- Node.js module for MongoDB connection with Mongoose ORM
- Support with following environment variables:
  - ```MONGODB_URI```
  - ```MONGODB_AUTH_USER```
  - ```MONGODB_AUTH_PASS```
  - ```MONGODB_AUTH_DB```
